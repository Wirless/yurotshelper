Setting up Yurots to play

After downloading the required files and extracting/installing them we should navigate to our ots folder where yurots.exe is located and open config.lua

the ip is gonna be set to auto which we can either change or leave as is.

After that we need to run the 7.6 client of tibia and ip changer where in ipchanger window we will input 127.0.0.1

Now we can apply ip settings in ip changer and it will automatically hook it to any opened client.

After that we can run yurots.exe and in client click enter game and type in 111111 as our account number and tibia as password.

After that we will see our character list we can go to GM Yurez.

with the GM Character we can test mechanics of the game like quests/npcs and spawn monsters and things.

basic commands that every gamemaster should know are:

/a number teleport x amount of tiles forward ( easy to get behind walls etc)

/b message broadcast message with red text on middle of screen for all active players

/m monstername spawn a monster in front of us

/i itemid create an item underneat our gamemaster character ( some newer engines spawn items in backpacks)

/n npcname spawn an npc in front of us that is scripted ( non scripted npcs will not spawn data/npc)
